Test Talimatları
Dashboard Test
  1. Admin paneline gidin
  2. Dashboard'un otomatik yüklendiğini kontrol edin
  3. Görevler bölümüne gidin
  4. Dashboard'a geri dönün
Ekleme Test
  1. Yeni kullanıcı ekleyin
  2. Yeni mağaza ekleyin
  3. Yeni kanal ekleyin
  4. Listelerde görünüp görünmediğini kontrol edin
Export Test
  1. Görevler bölümüne gidin
  2. "Görevler Excel" butonuna tıklayın
  3. Excel dosyasını indirin
  4. Açıklama kolonunu kontrol edin
